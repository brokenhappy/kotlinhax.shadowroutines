# Coroutines integration

This directory contains modules that provide integration with various asynchronous callback- and future-based libraries.
Module names below correspond to the artifact names in Maven/Gradle.

## Modules

* [kotlinhax-shadowroutines-guava](kotlinhax-shadowroutines-guava/README.md) -- integration with Guava [ListenableFuture](https://github.com/google/guava/wiki/ListenableFutureExplained).
* [kotlinhax-shadowroutines-slf4j](kotlinhax-shadowroutines-slf4j/README.md) -- integration with SLF4J [MDC](https://logback.qos.ch/manual/mdc.html).
* [kotlinhax-shadowroutines-play-services](kotlinhax-shadowroutines-play-services) -- integration with Google Play Services [Tasks API](https://developers.google.com/android/guides/tasks).
